# RealtimeChart
Restfull api with node js + express js and token authentic for accessing with middleware. token -> /api/auth/token

## Installitation

```
npm install
```

## Running

```
npm run dev
```

It will show output :

```
Open http://localhost:1999
```

open this with http:://localhost:3000 on your browser.
